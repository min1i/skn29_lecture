| 변수명 | 기호 | 정의 및 역할 | 권장 기본값 |
| :--- | :---: | :--- | :--- |
| **Gradient** | $\nabla_\theta J(\theta)$ | 현재 단계에서 계산된 기울기(미분값)입니다. | - |
| **Learning Rate** | $\eta$ (eta) | 가중치를 업데이트할 보폭을 결정하는 학습률입니다. | $0.001$ |
| **1st Moment** | $m_t$ | 기울기의 지수 이동 평균(Momentum)으로 방향을 결정합니다. | - |
| **2nd Moment** | $v_t$ | 기울기 제곱의 지수 이동 평균(RMSProp)으로 보폭을 조절합니다. | - |
| **Beta 1** | $\beta_1$ | 1차 모멘트(방향)를 얼마나 유지할지 결정하는 계수입니다. | $0.9$ |
| **Beta 2** | $\beta_2$ | 2차 모멘트(보폭)의 영향력을 조절하는 계수입니다. | $0.999$ |
| **Epsilon** | $\epsilon$ | 수식에서 분모가 0이 되는 것을 방지하는 아주 작은 값입니다. | $10^{-8}$ |
